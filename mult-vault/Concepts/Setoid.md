# Setoid

A [[Setoid]] is a type together with an equivalence relation.

Example:

```text
type: pairs of natural numbers
relation: (a, b) is equivalent to (c, d) when a + d = c + b
```

Instead of forming a new quotient type, a proof can carry the relation and show that functions respect it.

Setoids are a common way to reason "up to equivalence."
