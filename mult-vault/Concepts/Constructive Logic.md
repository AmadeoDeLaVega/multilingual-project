# Constructive Logic

[[Constructive Logic]] treats a proof as evidence that can be built.

Example:

```text
To prove "there exists an x with property P",
constructive logic asks for a concrete x and a proof that P x holds.
```

Classical logic may prove existence indirectly, for example by contradiction, without producing the witness immediately.

Rocq is constructive by default, while Lean libraries often enable classical reasoning for mathematics.
