# Quotient Type

A [[Quotient Type]] groups values by an equivalence relation.

Example:

```text
integers = pairs (a, b) of natural numbers
(a, b) is equivalent to (c, d) when a + d = c + b
```

Here `(3, 1)` and `(5, 3)` both represent the integer `2`, so the quotient type treats them as the same value.

Quotients are convenient for mathematics because many objects are defined "up to equivalence."
