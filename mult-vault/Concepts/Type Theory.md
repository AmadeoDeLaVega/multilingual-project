# Type Theory

[[Type Theory]] is a foundation for mathematics and programming where every expression has a type.

Instead of saying only that a statement is true or false, type theory often asks whether a type has a term.

Example:

```text
2 : Nat
```

This means `2` is a term whose type is natural numbers.

For proof assistants, a proposition is also a type. Proving the proposition means constructing a term of that type.
